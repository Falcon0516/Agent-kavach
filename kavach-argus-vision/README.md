# KAVACH — AI-Agentic Women's Safety Platform
**Sentinel Hack 6.0 | KSIT Bengaluru | April 2026**
> "One word. Eight seconds. Six AI agents. Real police. Real WhatsApp. Real FIR."
## Architecture
- **Backend** (MSI): FastAPI + LangGraph 6-agent pipeline
- **Dashboard** (MacBook): React + Leaflet command center + police dashboard
- **Guardian PWA** (HP Pavilion / Android #3): React PWA with fake call, shake trigger
- **Call Bridge** (Mi Notebook / Windows): Phone Link outbound + incoming call handler
## Branches
